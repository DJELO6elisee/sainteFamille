🧹 Nettoyage du Cache

Si vous voyez encore l'ancienne version du site, utilisez ces boutons pour forcer la mise à jour :

Instructions pour les utilisateurs :

  1. Sur mobile : Vider le cache de votre navigateur dans les paramètres
  2. Sur Chrome : Ctrl+Shift+R (Windows) ou Cmd+Shift+R (Mac)
  3. Sur Safari : Cmd+Option+R
  4. Sur Firefox : Ctrl+F5 (Windows) ou Cmd+Shift+R (Mac)
